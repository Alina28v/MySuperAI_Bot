# 🤖 [MySuperAI_Bot](https://t.me/fast_ai_project_bot)

Це інтелектуальний Telegram-бот, який не просто відповідає на питання, а є персональним помічником у навчанні та технічних завданнях. Бот спеціалізується на темах УКД, Нейромереж та Програмування.

## 🎯 Цільова аудиторія

*   Студенти: яким потрібна допомога з освітніми матеріалами та швидкий доступ до бази знань.
*   Розробники: як приклад реалізації безпечного бота з використанням екологічних методів збереження ключів.
*   Допитливі: всі, хто хоче дізнатися більше про світ ШІ через зручний інтерфейс.

## 🚀 Технічна реалізація

Проєкт реалізований на Python з використанням сучасних паттернів розробки:

1.  Context Memory: Бот пам'ятає історію розмови, що дозволяє вести осмислений діалог.
2.  Security: Використання .env для захисту API-ключів (як показано у налаштуваннях).
3.  Command System: Підтримка команд управління, включаючи /clear для скидання контексту та /language.

## 🛠 Інструкція із запуску

1. Клонувати репозиторій:
   git clone https://github.com/Alina28v/MySuperAI_Bot.git
2. Налаштувати оточення:
   Створіть файл .env та додайте ваш BOT_TOKEN.
3. Запустити:
   python main.py
